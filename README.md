# Markdown AppNote

An Electron application with Vue + TypeScript + UnoCSS + Pinia + MdEditorv3

## Project Setup

```bash
git clone repo
```

### Install

```bash
$ pnpm install
```

### Development

```bash
$ pnpm dev
```

### Build

```bash
# For windows
$ pnpm build:win

# For macOS
$ pnpm build:mac

# For Linux
$ pnpm build:linux
```
